fibonacci#

project_template_python_pure.fibonacci(n)#

Calculate the nth fibonacci number.

Parameters:
nint

Which number to calculate, n >= 0.

Returns:
fib_nint

The nth fibonacci number

Examples

>>> fibonacci(0)
0
>>> fibonacci(1)
1
>>> fibonacci(2)
1
>>> fibonacci(3)
2
